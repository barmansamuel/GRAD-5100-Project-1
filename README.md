# GRAD-5100-Project-1
First project incorporating LLMs

This app gives specific advice for fighting games based on user input. Users are prompted for a game, a character, and a skill level and are given concise tips, tricks, and combos corresponding to the chosen skill level.

The LLM used is Ollama. This was chosen above other popular options like OpenAI or Gemini because it is open source and run locally.
